# Audio Files

This directory contains audio files for the application's immersive sound design.

## Required Audio Files

### Ghost Mode (Ouija Board)
- `drone.mp3` - Low frequency ambient drone for continuous background atmosphere
- `static_glitch.mp3` - Static/glitch sound effect when AI responds

### Legal Mode (Legal Eagle)
- `typewriter_fast.mp3` - Typewriter/keyboard click sound when AI responds

## Where to Get These Files

Download free, legal audio files from:
- **Freesound.org** (requires free account)
- **Pixabay.com** (no account needed)

### Recommended Search Terms

**For Ghost Mode:**
- "horror ambience"
- "low drone"
- "space hum"
- "static glitch"
- "radio interference"

**For Legal Mode:**
- "keyboard typing"
- "mechanical switch"
- "typewriter"
- "pen scratch"

## File Specifications

- Format: MP3
- Bitrate: 128kbps or higher
- Length: 
  - Background ambience: 30-60 seconds (will loop)
  - Response sounds: 1-3 seconds

## Volume Recommendations

- Background ambience: 30% volume (0.3)
- Response sounds: 50% volume (0.5)

These volumes are set in the JavaScript code but can be adjusted based on your specific audio files.
